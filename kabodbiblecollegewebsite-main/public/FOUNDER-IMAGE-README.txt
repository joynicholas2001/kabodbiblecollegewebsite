FOUNDER IMAGE FOR ABOUT PAGE
============================

To show the founder photo on the About page:

1. Copy your founder photo file (the one in app/assets or elsewhere).
2. Paste it INTO THIS FOLDER (the "public" folder).
3. Name the file exactly:  founder.jpg
4. You should have:  public/founder.jpg
5. Restart the dev server (Ctrl+C then npm run dev).
6. Open http://localhost:3000/about and refresh.

The About page will load the image from /founder.jpg.

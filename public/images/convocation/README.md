# Convocation / Hero Carousel Images

Add your hero carousel images here. Download photos from the Convocation 24 folder and save them as:

- **slide1.jpg** (or .JPG)
- **slide2.jpg**
- **slide3.jpg**
- **slide4.jpg**, **slide5.jpg** (optional — add more for a longer carousel)

**Google Drive folder:** [Convocation 24](https://drive.google.com/drive/folders/1-59utTimM12ab0puECeZBPkBIywb_JVx)

1. Open the folder link above.
2. Select 3 or more images (e.g. DSC03436.JPG, DSC03441.JPG, DSC03452.JPG).
3. Download them.
4. Rename to `slide1.jpg`, `slide2.jpg`, `slide3.jpg` (and optionally `slide4.jpg`, `slide5.jpg`).
5. Place the files in this folder: `public/images/convocation/`.

The carousel will use these as full-width background images with the Kabod Bible College text overlaid. If a file is missing, that slide will show a purple gradient instead.
